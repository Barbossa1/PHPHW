<?php
    $a = 5;
    $b = '05';
    var_dump($a == $b);         // Почему true?
    var_dump((int)'012345');     // Почему 12345?
    var_dump((float)123.0 === (int)123.0); // Почему false?
    var_dump((int)0 === (int)'hello, world'); // Почему true?
?>

Почему true?
$а является числом, значит $b тоже приводится к числу;
Почему 12345?
вручную привели к типу int
Почему false?
когда любой из операндов является числом с плавающей точкой,
все операнды интерпретируются как числа с плавающей точкой
Почему true?
true принимает значение 'hello, world', те 1, а false – 0;
